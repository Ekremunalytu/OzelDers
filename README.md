### Hafta 1 - Python Temelleri
İlk hafta, Python programlamanın temellerine odaklanmaktadır. Öğrencilerin pratik yapabilmesi için `Hafta 1/odev_ders1.ipynb` dosyası içerisinde çeşitli zorluk seviyelerinde 12 görevli ev ödevi bulunmaktadır.

- **Bölüm 1:** `print()` fonksiyonu ve veri tipleri/değişkenler (str, int, float, bool)
- **Bölüm 2:** Temel matematiksel işlemler (+, -, *, /, //, %, **)
- **Bölüm 3:** Mantıksal koşul yapıları (if / elif / else)
- **Bölüm 4:** Listeler (append, remove, index, len vb. metodlar)
- **Bölüm 5:** Fonksiyonlar (def, return, parametre kullanımı)

**Ödev Zorluk Dağılımı:** Kolay (6 Görev), Orta (5 Görev), Zor/Bonus (1 Görev)
