# Ozel Ders Materyalleri

Bu klasorde lise seviyesinde yapay zeka giris dersi icin hazirlanan materyaller bulunur.

Onerilen baslangic dosyalari:

- `lise_ai_dersi_kodsuz.ipynb`: Kod gostermeden, sohbet ve anlatim temelli ilk ders notebook'u
- `lise_ai_dersi_sade.ipynb`: Lise ogrencisi icin sade, tek derste anlatilabilir notebook
- `ders_plani_ai_giris.md`: Ogretmen icin kisa akis, anlatim notlari ve odak noktasi
- `mufredat_2_ders_ai.md`: Ilk iki ders icin net konu plani ve hedefler

Mevcut notebooklar:

- `lise_ai_dersi.ipynb`: Kisa onceki surum
- `ai_lise_dersi_genis.ipynb`: Daha genis ama ilk ders icin fazla yogun surum
